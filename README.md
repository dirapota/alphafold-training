# AlphaFold Training Course

Course for the SIB AlphaFold training, covering protein structure prediction with AlphaFold2 and AlphaFold3 using ColabFold and AlphaFold Server.

**Course website:** [https://dirapota.github.io/alphafold-training/](https://dirapota.github.io/alphafold-training/)

## Course content

- **Day 1**
  - ColabFold tutorial (monomer, custom MSA, multimer, alternative conformations)
  - Mol\* tutorial (structure visualization and analysis)
  - Slides

- **Day 2**
  - AlphaFold Server tutorial and Boltz-2 (apo vs Ca²⁺-bound calmodulin, protein-ligand complex with Boltz-2)
  - Slides
